# Example API gateway

## Goal

Create 3 services
* Service 1
* Service 2
* API gateway

## Steps

1. `npm install` and `npm start`
2. Call http://localhost:3001/
3. Call http://localhost:3001/service1
4. Call http://localhost:3001/service2
5. Call `curl -H "Content-Type:application/xml" http://localhost:3001`
